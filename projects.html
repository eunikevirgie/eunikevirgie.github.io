“New Normal”: Singapore’s Tourism Recovery After COVID-19

by Virgie Yuliana

HOME | LinkedIn | GitHub

Aims

This project examines how Singapore’s tourism sector recovered from the COVID-19 shock between 2019 and 2024, and whether the structure of visitor markets has returned to its pre-pandemic pattern or shifted to a new normal.

Project Overview

International travel to Singapore collapsed in 2020 following global border closures. While visitor numbers have since rebounded, recovery has varied substantially across source markets. Using monthly arrival data, this project visualizes the depth of the pandemic shock, compares recovery paths across visitor markets, and highlights changes in tourism composition during the post-COVID period.

Interactive charts allow users to explore recovery dynamics over time and across markets, providing insights into resilience, dependence, and structural change in Singapore’s tourism economy.

Data and Approach

The analysis uses monthly international visitor arrival data from Singapore’s Department of Statistics (Singapore DOS). Data are cleaned and processed using Python, and interactive visualisations are built with Vega-Lite. All processing steps are automated to allow straightforward updates as new data become available.
